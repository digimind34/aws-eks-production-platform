# AWS EKS Production Platform

This project demonstrates a production-style Kubernetes platform on AWS using Amazon EKS, Terraform, GitHub Actions, and Kubernetes manifests.

## Project Goals

- Provision a secure AWS VPC using Terraform
- Deploy an Amazon EKS cluster
- Configure managed node groups
- Deploy a containerized sample application
- Expose the app using Kubernetes services and ingress
- Add CI/CD with GitHub Actions
- Document the architecture like a real DevOps production project

## Tech Stack

- AWS
- Amazon EKS
- Terraform
- Kubernetes
- Docker
- GitHub Actions
- IAM
- VPC
- Load Balancer / Ingress
